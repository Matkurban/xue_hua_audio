# xue_hua_audio_player_example

Demo app for the [xue_hua_audio_player](../README.md) plugin (^1.0.0).

**English** | [插件中文文档](../README.zh-CN.md)

## Run

```bash
flutter run
```

## Tabs

- **Recording** — microphone device selection, record/pause/stop, level meter, playback with progress stream
- **Playback** — multi-track asset playback, volume, seek, progress streams

## Tests

```bash
flutter test integration_test/playback_test.dart -d macos
```

See the [main README](../README.md) for full API and platform setup.
